# Dataloader

Dataloader 는 성능 개선을 위한 도구로, 일괄처리 및 캐싱을 이용하여 요청에 대한 비용을 줄이는 역할을 수행합니다.

Graphql 에서는 N+1 문제를 해결하기 위해 사용합니다.
